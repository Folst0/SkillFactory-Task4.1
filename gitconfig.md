### Команда Gitconfig

Команда ***gitconfig*** используется для настройки параметров отображения важной информации, которая будет обозначать создателя проекта или иных репозиториев и его данные для обратной связи,     чтобы оперативно понимать кто на каком уровне что сделал.

### К такой информации относятся: 
    -Ваше имя 
    -Адресс электронной почты
---
 ### Примеры использования команды **gitconfig** :
#### Указание своего имени:
```bash=
git config --global user.name "Your name" 
```
---
#### Указание своей почты:
```
git config --global user.email "your_email@whatever.com"
```
