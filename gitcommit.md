# Git commit

*Git commit* - команда, при исспользовании которой все новые изменения сохраняются в репозиторий.

При ее использовании необходимо указать сообщение, что именно изменилось,чтобы было легче ориентироваться в изменениях проекта.

Для этого используется параметр **-m**

При написании комментария ориентируются на командные правила, но при большинствах проектах стараются излагать **коротко, отражая главную суть**, чтобы все могли понимать что изменилось без лишних усилий.

---
## Пример:

`git commit -m "Add new file"`
