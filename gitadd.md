# Git add

*Git add* - это комманда которая позволяет изменять из рабочего каталога в область подготовленных фаилов(индекс)

Можно представить это в виде примера:

1. **Рабочий каталог** - кухня со всеми ингридиентами
2. `git add` - это приготовленные блюда которые вы подаете или добавляете в меню
3. `git commit` - Это полностью готовое меню с возможностью выбрать блюда
